# 宾馆管理系统 python1177

## 项目概述

宾馆管理系统是基于Python的Django框架与Vue前端技术构建的，它实现了宾馆日常业务管理的数字化、自动化。

## 技术栈

- 后端：Python、Django、MySQL
- 前端：Vue

## 功能模块

系统主要包括以下功能模块：

- 首页：展示宾馆基本信息与实时动态。
- 个人中心：管理个人信息。
- 会员管理：处理会员账户与等级相关事务。
- 客房类型管理：维护不同类型客房的信息。
- 热门客房管理：特别推荐与展示受欢迎的客房。
- 客房预约管理：处理客房预订与取消预订等操作。
- 系统管理：监控系统运行状态，进行基础设置。

## 系统角色

- 系统管理员：负责整体系统的维护与管理工作。
- 前台服务人员：处理日常业务操作，如会员服务、预订管理等。
- 客户：可查看房间信息并预约房间。

## 运行环境

- 操作系统：支持主流操作系统
- 后端：Python 3.x、Django 2.x、MySQL 5.x
- 前端：Vue.js 2.x

1. 环境准备：确保Python、MySQL、Node.js等开发环境正确安装。
